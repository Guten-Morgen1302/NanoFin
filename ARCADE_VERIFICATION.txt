TROUBLESHOOTING GUIDE

If something doesn't work, check:

Games not loading:
  1. Run: npm run setup:arcade (copy games to public/)
  2. Verify: ls public/games/FinQUEST-main/
  3. Clear browser cache: Ctrl+Shift+R
  4. Check DevTools Network tab for 404 errors
  
Stats not updating:
  1. Check console for errors
  2. Verify localStorage enabled in browser
  3. Check that game exit function is called
  4. Try clearing localStorage and playing again
  
Modal won't close:
  1. Try ESC key
  2. Try clicking âœ• button
  3. Check console for JavaScript errors
  4. Try closing DevTools and retrying
  
Games look ugly/misaligned:
  1. Clear browser cache: Ctrl+Shift+R
  2. Check DevTools CSS for any conflicting styles
  3. Verify FinancialArcade.css loaded (check Network tab)
  4. Try different browser
  
Page won't load at all:
  1. Check dev server is running: npm run dev
  2. Check URL is correct: http://localhost:5173/arcade
  3. Check App.jsx has /arcade route
  4. Check console for import errors
